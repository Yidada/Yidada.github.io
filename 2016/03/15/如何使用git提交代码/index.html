如何通过git来提交代码

本文默认采用webstorm进行项目开发

首先搭建项目环境

  1. 下载git，windows地址：https://git-scm.com/download/
  2. 安装git，注意在安装过程中要将git添加到cmd中。
  3. 登录github注册一个帐号（如果登录不上，请翻墙）
  4. 安装WebStorm，自行下载安装。
  5. 在webstorm当中注意到
    如图
  6. 在VCS中选择Checkout from Version Control -> Github
  7. 在弹出框中输入我们的项目的github地址：
    https://github.com/Yidada/ife.git
  8. 之后会从项目代码仓库下载代码到WebtStorm本地。

编写代码并提交

项目下载下来之后。
项目结构如下：

  • 其中Decade-1表示第一阶段；
  • Task-1 表示任务一；

各自进入自己个的（jijige）的文件夹内，文件夹默认创建了assets文件夹和index.html文件，其中assets文件夹内包含了js、css、img文件夹用于存放javascript和css和图片文件。

当代码编写完成之后，提交代码的过程主要分两部：

  1. commit代码到本地仓库。
  2. push提交的代码到远程仓库。

所以当commit了之后一定要再选择push一次
commit可以点工具栏内的小图片

push则需手动进入VCS -> Git -> Push
或者push也可以采用快捷键（windwos下：ctrl+alt+k）进行push。

注：提交代码的时候最好在commit message中写上备注用于表示当前提交的内容
